# TAI Advisory

**Technology & Justice for a Digital Future**

TAI Advisory is a U.S.-based policy and research organization at the intersection of AI governance, digital rights, and public interest technology.

## Live Site

Deployed via GitHub Pages at: https://ajadonai.github.io/tai/

## Stack

- Pure HTML/CSS/JS — no build step
- Newsreader (serif) + Manrope (sans) typography
- Warm ivory palette with forest teal accent
- Fully responsive: mobile, tablet, desktop
- Safe viewport handling for iOS Safari/Chrome
